# Introduction to Cybersecurity - IFSP - Salto Campus

### About This Course

This repository documents my journey through the **Introduction to Cybersecurity Course** at IFSP (Instituto Federal de São Paulo) - Salto Campus, enriched with complementary in-depth study content.

Here you'll find **organized notes, technical analyses, and fundamental concepts** covering everything from basic security principles to advanced protection technologies.



**Structure**: 5 modules + final exam\
**Goal**: Build a solid foundation in cybersecurity for professional career development

* Learn about **fundamental online security concepts** and the impact of cybersecurity
* Understand the **most common cyber attacks**, vulnerabilities, and associated risks
* Discover how **companies defend against attacks** and implement protective measures
* Explore **information, resources, and opportunities** to build a career in cybersecurity
* &#x20;Build a **technical portfolio** documenting your learning journey

## Module Structure

Module 1: Introduction to Cybersecurity Fundamental concepts and general context

What is cybersecurity? Importance and impacts Career pathways

Module 2: Analyzing Cyber Attacks Types of malware, attack techniques, and vulnerabilities

Module 3: Protecting Data and Privacy Defense, encryption, and secure practices

Module 4: Devices and Security Technologies Corporate defense architecture

#### **Module 1: Introduction to Cybersecurity**

Fundamental concepts and general context

* What is cybersecurity?
* Importance and impacts
* Career pathways

#### **Module 2: Analyzing Cyber Attacks**

Types of malware, attack techniques, and vulnerabilities

| Topic                  | Coverage                                |
| ---------------------- | --------------------------------------- |
| **Malware**            | Types, symptoms, detection              |
| **Social Engineering** | Phishing, pretexting, tailgating        |
| **DoS/DDoS**           | Denial of service, botnets              |
| **On-Path Attacks**    | Man-in-the-Middle, interception         |
| **Password Attacks**   | Brute force, dictionary, rainbow tables |
| **Exploits**           | Hardware and software vulnerabilities   |

#### **Module 3: Protecting Data and Privacy**

Defense, encryption, and secure practices

| Topic                 | Coverage                            |
| --------------------- | ----------------------------------- |
| **Device Protection** | Firewall, antivirus, IoT security   |
| **Network Security**  | Secure Wi-Fi, VPN, WPA2             |
| **Strong Passwords**  | Passphrases, NIST guidelines        |
| **Data Maintenance**  | Encryption, backup, secure deletion |
| **Authentication**    | 2FA, OAuth, web privacy             |

#### **Module 4: Devices and Security Technologies**

Corporate defense architecture

| Topic                   | Coverage                                |
| ----------------------- | --------------------------------------- |
|  **Defense in Depth**   | Layered security approach               |
| **Security Components** | Routers, firewalls, IPS, VPN            |
| **Firewalls Detailed**  | Types, layers, NAT                      |
| **Port Scanning**       | Nmap, common ports, scanning techniques |
| **IDS/IPS**             | Intrusion detection and prevention      |

### How to Use This Repository?



1. Start with **Module 1** for fundamental concepts
2. Study **Module 2** to understand attacks
3. Learn **Module 3** for defense and protection
4. Deepen knowledge with **Module 4** on technologies
5. Use **Glossary** to look up terms

**Estimated time**: 20-30 hours of dedicated study
